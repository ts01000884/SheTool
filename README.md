# 我的工具箱 (My Toolbox)

這是一個集合各種實用前端小工具的專案，旨在提供快速、方便的解決方案來處理日常開發或營運中的瑣碎任務。

## 目前工具

### [推廣網址產生器](./url-generator/index.html)

一個用於快速產生行銷推廣網址的工具。

**主要功能：**
- **參數附加**：輕鬆為指定的網址附加上 `sales_code` (推廣編號) 和 `campaign_code` (活動編號)。
- **保留現有參數**：產生新網址時，會自動保留原始網址中已存在的查詢參數。
- **智慧記憶**：會自動記住上次使用的 `sales_code` 和 `campaign_code`，減少重複輸入。
- **歷史紀錄**：在瀏覽器本機儲存最近產生過的網址，方便快速取用。
- **協作分享**：可以產生一個預先填好「分享網址」和「活動編號」的工具連結，方便分享給同事填寫各自的推廣編號。
- **URL 預填**：支援透過工具本身的網址參數來預先填寫所有欄位。

## 專案結構

本專案採用模組化結構，每個工具都存放在獨立的子目錄中，方便未來擴充。

```
/
├── README.md                (本說明檔案)
├── index.html               (工具箱首頁)
├── style.css                (首頁樣式)
└── url-generator/           (推廣網址產生器工具)
    ├── index.html
    ├── style.css
    └── script.js
```

## 如何使用

1.  **本地使用**：
    -   Clone 或下載本專案。
    -   直接使用網頁瀏覽器開啟根目錄的 `index.html` 檔案即可看到工具首頁。

2.  **線上部署**：
    -   將整個專案部署到任何靜態網站託管服務，例如 GitHub Pages, Netlify, Vercel 等。

## 如何擴充

您可以輕鬆地在這個專案中加入您自己的新工具：

1.  在根目錄下建立一個代表新工具的資料夾 (例如 `new-tool/`)。
2.  在新資料夾中開發您的工具，可以包含 `index.html`, `style.css`, `script.js` 等檔案。
3.  開發完成後，編輯根目錄的 `index.html`，在 `<main class="tool-grid">` 中仿照現有格式，新增一個指向您新工具的 `<a>` 連結卡片即可。
